Čo je to Java?

Java je vysokoúrovňový, univerzálny, pamäťovo bezpečný, objektovo orientovaný programovací jazyk. Je to objektovo orientovaný jazyk, ktorý sa riadi heslom: "Write once, run anywhere"

Projekt jazyka Java v júni 1991. Java bola pôvodne navrhnutá pre interaktívnu televíziu, ale v tom čase bola príliš pokročilá pre odvetvie digitálnej káblovej televízie. Jazyk sa pôvodne volal Oak podľa duba, ktorý stál pred Goslingovou kanceláriou. Neskôr projekt niesol názov Green a nakoniec bol premenovaný na Java, podľa názvu Java coffee, druhu kávy z Indonézie. Gosling navrhol Javu so syntaxou v štýle C/C++, ktorá bola systémovým a aplikačným programátorom známa.

Na týchto stránkach nájdete môj osobný plán vzdelávania, kde rozoberám základné koncepty, ktoré musí každý budúci Java programátor ovládnuť.
